# C# ToDoList
ToDoList console application with C#

## Ana Özellikler
- Kart Ekle
- Kart Güncelle
- Kart Sil
- Kart Taşı
- Board Listeleme


### Kart
```
Kart İçeriği:
Baslık
Icerik
Atanan Kisi (Takım üyelerişnden biri olmalı)
Büyüklük (XS, S, M, L, XL)
```
